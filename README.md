## Prerequisites

- Node.js v22
- pnpm

## Installation

1. Clone the repository
2. Install dependencies:

```bash
cd challenge-scale-ai
nvm use
pnpm install
```

## Running the application

1. Build the application:

```bash
pnpm build
pnpm start
```

2. Go to http://localhost:3000 to see the application.
